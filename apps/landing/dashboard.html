Getting Started Guide

Everything you need to know about the platform

1. Creating Your First Agent

There are 3 ways to create an agent:

Constructor

Visual drag-and-drop builder.

AI Chat

Describe what you need in natural language.

Telegram Bot

Send a message to @TonAgentPlatformBot

2. Flow Builder (Constructor)

Controls:

  • Left-click on empty space — move camera
  • Mouse wheel — zoom in/out
  • Click node — select & configure
  • Drag from output port to input port — connect
  • Click on connection + Delete — disconnect
  • Right-click on connection — delete it
  • Deploy button — create agent from flow
  • Ctrl+Z / Ctrl+Y — undo / redo

Block categories: Triggers, TON, Gifts, Web, Telegram, Logic, State, Output

3. AI Assistant Chat

Click the chat button in the bottom-right corner.

  • Create agents from descriptions
  • Help with constructor and settings
  • Answer questions about the platform
  • Analyze NFT and gift prices

Tip: Chats sync between studio and Telegram!

4. Marketplace

Browse ready-made agent templates by category.

Click 'Install' to add a template.

5. TON Integration

Connect your TON wallet via TonConnect.

6. Settings & Persona

Configure your AI provider and API keys in Settings.

Set up your Persona to customize the AI.
